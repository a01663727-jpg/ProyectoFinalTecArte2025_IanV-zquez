Este proyecto consiste de un programa que calcula el área de diferentes figuras como: triángulos, círculos y rectángulos; a partir de la información sobre sus dimensiones que se proporciona en un archivo .csv 
Los pasos generales que se siguieron para completar el reto fueron los siguientes:
1.- creamos una carpeta especial para guardar los archivos del proyecto con el nombre: proyectoFinalArte. 
2.- dentro de esta carpeta, creamos un archivo llamado figuras.csv donde copiamos las dimensiones de 100 figuras distintas.
3.- creamos una subcarpeta dentro de la carpeta del proyecto para guardar todas las librerías necesarias para ejecutar el programa. 
5.- creamos el programa en Python, llamado figurasPrincipal.py a través de la terminal. 
6.- creamos un segundo programa de Python que contiene las funciones necesarias para calcular el área de cada una de las figuras, este programa debe estar en la carpeta del proyecto para poderse vincular. 
7.- creamos un archivo de documentación que explica el proyecto y como ejecutarlo.
7.- pasamos los archivos al área de stage con el comando "git add" 
8.- pasamos los archivos al repositorio local con "git commit -m "mensaje""
9.- pasamos los archivos al repositorio remoto (GitHub).



Para ejecutar el programa escribe: 
python3 figurasPrincipal.py
